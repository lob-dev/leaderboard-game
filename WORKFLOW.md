# Workflow - Leaderboard Game

## Autonomous Work Rules
- Lobster works continuously on tasks from Trello board
- Pick up To Do items, move to Doing, work them, move to Done
- Commit to git after each meaningful chunk of work
- Only ping Yotam when BLOCKED (need install, decision, access, etc.)

## Ping Yotam When:
- Need something installed on the machine
- Need a creative/design decision (game direction, art style, etc.)
- Need credentials or access to something
- Hit a bug that requires human intervention
- Task is done and ready for review

## Don't Ping Yotam When:
- Working through normal code tasks
- Making routine commits
- Moving cards on Trello
- Fixing your own bugs
