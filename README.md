# Leaderboard Game - POC

**Concept:** The leaderboard IS the game. It's front and center at all times. Your goal is to climb it.

## Vision
- The leaderboard is not a menu screen — it's the main gameplay view
- Players see their position and compete to climb in real-time
- Every action feeds back into your rank
- Simple, addictive, competitive

## Tech Stack
- Unity 2022 LTS+
- C#
- Git for version control

## Project Structure
```
Assets/
  Scripts/
    Leaderboard/     - Core leaderboard system
    Player/          - Player data & actions
    UI/              - UI components
    Networking/      - Future multiplayer support
  Scenes/
    MainScene.unity  - The one scene (leaderboard IS the game)
  Prefabs/
  Art/
```

## Getting Started
1. Install Unity Hub + Unity 2022 LTS (or Unity 6)
2. Open this folder as a Unity project
3. Open `Assets/Scenes/MainScene.unity`
4. Hit Play
